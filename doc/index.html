PencilBox

概要

PencilBoxは，(株)ニコリ発行の雑誌「パズル通信ニコリ」に掲載されている各種ペンシルパズルを，鉛筆をマウスに持ち替えてパソコン上で遊んだり作ったりするためのプログラムです。

以下の19種類のペンシルパズルに対応しています。

カックロ， 黒どこ， 碁石ひろい， 四角に切れ， 数独， スラローム， スリザーリンク， 天体ショー， ナンバーリンク， ぬりかべ， 波及効果， 橋をかけろ， 美術館， ひとりにしてくれ， フィルオミノ， へやわけ， ましゅ， ヤジリン， ＬＩＴＳ

動作環境

PencilBox の実行には Java Runtime Environment のversion1.5以上が必要です。 Java Runtime Environment は http://java.com/ より入手できます。

作者は Windows 上で使用しています。 他の環境でも動作することを期待していますが，特に動作確認は行っていません。
